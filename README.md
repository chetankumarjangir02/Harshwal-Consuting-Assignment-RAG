#  Harshwal Consulting Assignment:-

This project is a **PDF Question Answering Application** built with **Streamlit** and **LangChain**.  
It allows users to **upload a PDF file** and **ask questions** about its content using **Azure OpenAI Embeddings** and **Google Gemini** for intelligent retrieval and response generation.

# Install Dependencies
pip install -r requirements.txt

# Run the App
streamlit run app.py
